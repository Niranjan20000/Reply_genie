**AI-Powered Email Generator using Gemini 2.0 Flash**

ReplyGenie is a modern web application that helps users **generate professional, casual, or friendly email replies** in seconds. Powered by **Google Gemini 2.0 Flash AI**, it provides **human-like, context-aware responses** with a sleek and responsive UI.

- Supports large email content inputs.
